## Compatibiltiy

ECMAScript 3 (1999.12),

* Firefox 1 + JavaScript 1.5
* Chrome 1
* Safari 3
* IE 5.5 + Jscript 5.5 ~ IE 8 + JScript 5.8 (2009.03)

ECMAScript 5 (2009.12)

* Firefox 4 + JavaScript 1.8.5
* Chrome 7
* Safari 5
* IE 9 (2011.03)
